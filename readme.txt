This is an AI based Nutrition deficiency prediction model.

To run this You need to first start the streamlit codes as follows one by one.

1.Report analysis.py
2.Data_Analytics.py
3.predictor.py

command: streamlit run app_name.py

Now you can run the flask app app.py.This provides a web interface with which we can see all our features.
In the navigation bar you can see all the features enabled.

Note: predictor.py is highly confidential as it is using google fit api .The authorisation is only for specified users only.The chatbot and Report Analysis works properly.
